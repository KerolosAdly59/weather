# 🌤️ Weather App

A modern **Weather Application** built with **HTML, CSS, JavaScript, and Bootstrap** that fetches real-time weather data from a **Global Weather API**.
Users can check weather conditions for their current location or any city worldwide.

---

## 🚀 Live Demo

👉 https://weather-phi-smoky-45.vercel.app/

---

## 🎯 Overview

This Weather App fetches live weather data from a reliable **Global Weather API**.
It demonstrates working with APIs, handling asynchronous requests, and building a clean, responsive UI using **Bootstrap** for layout and styling.

---

## ✨ Key Features

* 🌍 Real-time weather data for any location
* 📌 Detects user location using global IP or browser geolocation
* 🌡️ Displays temperature, humidity, wind speed, and weather description
* 📱 Fully responsive using **Bootstrap**
* ⚡ Smooth and interactive interface

---

## 🛠️ Tech Stack

* 🟦 HTML5
* 🎨 CSS3 + Bootstrap 5
* ⚡ Vanilla JavaScript (ES6+)
* 🌐 Global Weather API (Real-time data)
* 🚀 Vercel (Deployment)

## ⚙️ How to Run Locally

1. Clone the repository:

```bash id="y8p4qf"
git clone https://github.com/KerolosAdly59/weather-app.git
```

2. Navigate to the project folder:

```bash id="k4t7vr"
cd weather-app
```

3. Open `index.html` in your browser (no server required for basic functionality)

> ⚠️ Make sure your API key is correctly set in `script.js` if required by the Global Weather API.
